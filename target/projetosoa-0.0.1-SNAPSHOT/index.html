Calculadora SOAP Service Test

Access the WSDL at: /projetosoa/ws/calculadora?wsdl

SOAP Request Example:

POST /projetosoa/ws/calculadora HTTP/1.1
Content-Type: text/xml; charset=utf-8
SOAPAction: "somar"

<?xml version="1.0" encoding="UTF-8"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/"
               xmlns:tns="http://soap/">
  <soap:Header/>
  <soap:Body>
    <tns:somar>
      <numero1>5</numero1>
      <numero2>3</numero2>
    </tns:somar>
  </soap:Body>
</soap:Envelope>

Expected Response:

<?xml version="1.0" encoding="UTF-8"?>
<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
  <soap:Body>
    <ns2:somarResponse xmlns:ns2="http://soap/">
      <return>8</return>
    </ns2:somarResponse>
  </soap:Body>
</soap:Envelope>
